# Cyber Security Diploma Labs

This repository contains practical cybersecurity lab exercises completed as part of my cybersecurity diploma training.

All testing activities were performed in authorized and controlled lab environments created for educational purposes.

---

## Lab Categories

### 1. SQL Injection (SQLi)
- Authentication bypass techniques
- Manual SQL payload testing
- Automated SQL injection tools practice
- Lab-based exploitation screenshots

### 2. Cross-Site Scripting (XSS)
- Reflected XSS
- Stored XSS
- Payload crafting and input validation testing

### 3. Local File Inclusion (LFI)
- File path manipulation
- LFI detection methods
- LFI to RCE concept demonstration (lab only)

### 4. Web Application Security
- Redirect behavior analysis
- WAF-protected lab testing
- Post exploitation simulation (controlled environment)

### 5. Network Security Assessment
- TCP Idle Scan research
- Vulnerability scanning
- Target IP assessment report (authorized lab systems)

---

## Tools Used

- Burp Suite
- Havij (SQLi automation – lab environment)
- Acunetix
- Manual HTTP request analysis
- Basic enumeration tools

---

## Reports Included

- Vulnerability Analysis Reports
- VAPT Assessment Report
- Lab Write ups with Proof of Concept
- Security Risk Classification

---

## ⚖ Ethical Statement

All activities documented in this repository were conducted in controlled lab environments or intentionally vulnerable systems for educational purposes only.

No unauthorized real-world systems were targeted.
